Grocery Web Application (MERN)

This project is a single-seller, multi-buyer grocery web application developed using the MERN stack as part of the 4th semester academic project.

# Features
- User registration and login
- Browse grocery products
- Add products to cart
- Place orders
- Seller can add, update, and delete products
- Order management for seller

# Tech Stack
- Frontend: React.js
- Backend: Node.js, Express.js
- Database: MongoDB
- Authentication: JWT / Session-based (if used)

# Project Type
- Academic Project (4th Semester)
- Single Seller, Multiple Buyers System
